# Developer Issue Tracker

This repository uses GitHub Issues to track bugs, enhancements, and development tasks.

For contributors, the source of truth is the GitHub issue board:

- https://github.com/Samuel1-ona/Hunty-contract/issues

## Notes

- Use the `.github/ISSUE_TEMPLATE/task.md` template for creating new development tasks.
- When issues are assigned, link feature branches or PRs back to the corresponding issue.
- If the project later adopts a local issue list, this file can be updated with issue summaries and expected work.
